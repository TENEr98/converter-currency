# converter-currency
Test-express24

# Installation
```sh
git clone https://github.com/TENEr98/converter-currency
cd converter-currency
```

# Configuration
Create file .env.local
Paste Token and Base Url into file

- VITE_TOKEN_KEY=
- VITE_BASE_URL=

# Run
```sh
yarn
yarn dev
```

# Build
```sh
yarn build
```
